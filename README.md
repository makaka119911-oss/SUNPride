# SUNPRIDE - Сайт эксклюзивных рукодельных работ

## Описание проекта
SUNPRIDE - это веб-сайт для демонстрации и продажи эксклюзивных рукодельных работ от мастера Татьяны Солнечной. Сайт выполнен в премиальном стиле с использованием темных золотых оттенков и анимаций.

## Особенности
- Полностью адаптивный дизайн для всех устройств
- Премиальный интерфейс с золотыми градиентами
- Интерактивная галерея товаров
- Модальные окна для детального просмотра товаров
- Плавные анимации и переходы
- Оптимизированная типографика для лучшей читаемости

## Технологии
- HTML5
- CSS3 с использованием CSS Variables
- JavaScript (нативный)
- Font Awesome иконки
- Google Fonts (Playfair Display, Montserrat)
